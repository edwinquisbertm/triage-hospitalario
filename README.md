# triage-hospitalario
Sistema de gestión de urgencias hospitalarias que prioriza pacientes por gravedad y no por llegada. Asigna triage de 1 a 5 (1 crítico, 5 leve), ordena automáticamente la cola, gestiona camas por área (Urgencias, UCI, Observación), permite búsqueda por cédula y conserva historial en archivos.
